# SkillSense 🎯

> **A High-Performance, AI-Driven Full-Stack Web Application**
> 
> 🌐 **Live Application:** [https://skill-sense-psi.vercel.app/](https://skill-sense-psi.vercel.app/)

SkillSense is a complex, modern web application that bridges the gap between Artificial Intelligence and Full-Stack Engineering. This project serves as a comprehensive showcase of my ability to architect scalable systems, write secure backend APIs, and design highly interactive user interfaces from the ground up.

---

## 📈 Engineering Impact & Highlights

This project was engineered to demonstrate proficiency in modern software development, focusing on core attributes that drive value in top engineering teams:

- 🧠 **Advanced AI Integration:** Successfully integrated complex Natural Language Processing (NLP) embeddings natively using `Transformers.js`. This runs entirely independent of expensive third-party APIs, demonstrating cost-conscious, high-privacy software engineering.
- ⚡ **High-Performance Architecture:** Engineered asynchronous background processing for heavy machine-learning tasks and PDF parsing, ensuring the frontend UI remains blazing-fast and highly responsive even under load.
- 🔐 **Robust Security Practices:** Implemented industry-standard JWT authentication, bcrypt password hashing, secure environment configurations, and role-based access control to protect sensitive user records.
- 📊 **Product-Minded UI/UX:** Designed and developed a sleek, professional dashboard using React, TailwindCSS, and Recharts to translate complex datasets into intuitive, interactive visual charts.
- 🏗️ **Scalable API Design:** Built a clean, RESTful Node.js/Express backend integrated with MongoDB, successfully handling layered database aggregations and complex relational queries.

## ✨ Key Technical Achievements

- **Unstructured Data Extraction:** Engineered seamless PDF parsing backend logic to extract, sanitize, and structure raw text data.
- **Algorithmic Data Normalization:** Wrote algorithms to normalize over 90+ different input aliases, ensuring extremely high calculation accuracy across the platform.
- **Complex State Management:** Utilized advanced React methodologies including Context APIs, custom hooks, and dynamic data fetching logic.
- **Batch Processing Pipelines:** Implemented optimized multi-document database pipelines for bulk operations, minimizing server load and database read/writes.
- **Full-Stack Deployment:** Successfully deployed and bridged a Vite-based single-page application with a decoupled machine-learning Express API across Vercel and Render.

## 🛠️ Technology Stack

- **Frontend:** React 18, Vite, Tailwind CSS 4, React Router, Recharts, Axios.
- **Backend:** Node.js, Express.js, MongoDB (Mongoose), Multer, PDF-Parse.
- **Artificial Intelligence:** Local NLP embeddings (`Xenova/all-MiniLM-L6-v2`).
- **Cloud & Deployment:** Vercel (Frontend Hosting), Render (Backend Pipeline).
